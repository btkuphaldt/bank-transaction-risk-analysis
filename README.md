# bank-transaction-risk-analysis
Bank Transaction Risk Analysis Dashboard

This project analyzes 2,512 banking transactions to identify behavioral patterns and potential fraud risk using rule-based logic.

Objective :

To explore transaction behavior, identify anomalies, and visualize risk indicators across different transaction types, amounts, and time periods.

Tools Used :
Excel (data preparation)
Key Features :
Interactive dashboard with filters and actions
Transaction type distribution (Credit vs Debit)
Transaction amount analysis
Login attempt behavior tracking
Time-based transaction trends
Rule-based Risk Flag system
Risk Logic

Transactions were flagged as risky based on:

High login attempts
High transaction amounts
Channel activity

Key Insight :

Fraud indicators are rare but concentrated in specific behavioral patterns, highlighting the importance of multi-factor risk detection rather than single-variable analysis.
